README 
Bij deze oplossing horen 2 PHP bestanden, die moet je op een local server toegankelijk maken. Bijvoorbeeld in localhost met Xamp
De fetchBAsic laat een aantal maneren zien om met javascript data op te vragen aan de PHP bestanden. Dat zijn API's
Dit zijn basic oplossingen, in een professionele omgeving is het uitgebreider en complexer. 
-- de APIs geven tekst terug, dat zou normaal een JSON bestand zijn
-- de APIs zullen meestal data opvragen bij een database aan de hand van de input, dat is in deze versie niet gemaakt, want dat maakt de backend flink complexer.
++ de APIs checken wel de input en je kunt verschl zien tussen GET en POST
